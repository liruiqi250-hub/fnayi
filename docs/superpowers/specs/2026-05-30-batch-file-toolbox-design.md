# 批量文件工具箱第一版设计

## 目标

做一个 Windows 桌面 EXE 软件，给没有编程基础的用户使用。用户打开软件后，通过按钮选择功能，选择文件或文件夹，点击开始处理，软件自动导出处理后的文件。

第一版重点是把日常办公文件处理跑通：Word 翻译、Excel 翻译/整理、PDF 转文字/翻译、文件批量重命名和整理。翻译能力使用 DeepSeek API，从项目 `.env` 读取密钥和模型配置。

## 第一版功能

### Word 翻译成英文

- 输入：一个或多个 `.docx` 文件。
- 操作：提取文档中的段落、标题、表格文字，调用 DeepSeek 翻译成英文。
- 输出：新的 `.docx` 文件。
- 文件名：原文件名后加 `_EN`，例如 `说明书.docx` 输出 `说明书_EN.docx`。
- 输出目录：默认在原文件旁边创建 `translated` 文件夹。

第一版优先保留正文、段落、标题和表格文字。复杂排版、页眉页脚、文本框、图片中文字识别作为后续版本增强。

### Excel 翻译/整理

- 输入：一个或多个 `.xlsx` 文件。
- 操作：
  - 翻译指定列中的文字。
  - 可选整理常见空行、空格和重复文件名字段。
- 输出：新的 `.xlsx` 文件。
- 文件名：原文件名后加 `_processed` 或 `_EN`。

第一版先提供简单选项：全表翻译、指定列翻译、仅整理不翻译。

### PDF 转文字/翻译

- 输入：一个或多个 `.pdf` 文件。
- 操作：
  - 从 PDF 提取可复制文字。
  - 调用 DeepSeek 翻译成英文。
- 输出：新的 `.docx` 文件。
- 文件名：原文件名后加 `_EN.docx`。

第一版处理“文字型 PDF”。扫描版 PDF 和图片型 PDF 需要 OCR，放到后续版本。

### 批量重命名/整理

- 输入：一个文件夹。
- 操作：
  - 按文件类型分类到 Word、Excel、PDF、Images、Other 文件夹。
  - 可选按日期加前缀。
  - 可选把空格替换成下划线。
- 输出：整理后的文件夹和一份处理报告。

第一版不自动删除源文件。默认复制到新输出目录，避免误伤原资料。

## 界面设计

软件主界面分两栏：

- 左侧：四个功能按钮，分别是 Word 翻译、Excel 翻译/整理、PDF 转文字/翻译、批量重命名/整理。
- 右侧：当前功能的操作区域，包括文件选择、处理选项、开始按钮、进度条和输出目录。

用户流程：

1. 打开 EXE。
2. 在左侧选择功能。
3. 点击选择文件或选择文件夹，也可以拖入文件。
4. 设置目标语言、输出位置、命名规则等简单选项。
5. 点击开始处理。
6. 处理完成后，软件显示输出文件夹按钮和处理报告。

## 技术方案

推荐使用 Python 开发桌面软件，并打包成 Windows EXE。

- 桌面界面：PySide6 或 CustomTkinter。
- Word 处理：`python-docx`。
- Excel 处理：`openpyxl`。
- PDF 文字提取：`pypdf` 或 `pdfplumber`。
- 图片和文件整理：Python 标准库 `pathlib`、`shutil`。
- API 调用：OpenAI 兼容客户端，`base_url=https://api.deepseek.com`。
- 打包：PyInstaller。

配置读取 `.env`：

- `DEEPSEEK_API_KEY`
- `DEEPSEEK_BASE_URL`
- `DEEPSEEK_MODEL`

## 数据流

1. 用户选择输入文件或文件夹。
2. 软件检查文件类型、大小和是否可读取。
3. 文件解析模块提取可处理文本。
4. 翻译模块把文本分段发送给 DeepSeek。
5. 输出模块生成新 Word、Excel 或整理后的文件夹。
6. 日志模块记录成功、失败和跳过的文件。
7. 界面显示进度和结果。

## 错误处理

软件必须避免让用户看不懂的报错。常见错误用中文提示：

- 没有配置 DeepSeek API Key：提示用户检查 `.env`。
- 网络连接失败：提示稍后重试或检查代理/网络。
- 文件被占用：提示关闭 Word、Excel 或 PDF 阅读器后重试。
- PDF 没有可提取文字：提示这是扫描版 PDF，第一版暂不支持。
- 单个文件失败：不中断整个批量任务，在报告中记录失败原因。

## 测试要求

第一版至少验证这些场景：

- 一个普通 Word 文件翻译后能生成英文 Word。
- 多个 Word 文件批量处理不互相覆盖。
- Excel 指定列翻译后其他列保留。
- 文字型 PDF 能导出英文 Word。
- 批量整理文件夹时不会删除源文件。
- API Key 缺失时显示中文提示。
- 网络失败时不会卡死界面。
